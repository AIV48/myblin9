# Prompt Engineering Notes

This document tracks prompt engineering methods used for AI idol content generation.

## Goals

- Maintain consistent character identity
- Preserve hairstyle, outfit, and facial structure
- Improve photorealistic rendering quality
- Reduce AI artifacts
- Support repeatable music video production

## Prompt Strategy

- Identity reference locking
- Scene and camera direction
- Lighting control
- Negative prompt refinement
- Multi-character consistency
